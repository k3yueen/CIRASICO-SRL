# CIRASICO SRL - Professional Website

A modern, responsive website for CIRASICO SRL, a leading provider of industrial scaffolding, thermal insulation, and anti-corrosion protection services in Romania.

## 🌟 Features

- **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX** - Professional construction industry aesthetic
- **Mobile Navigation** - Hamburger menu with smooth transitions
- **Multi-language Support** - Romanian and English content
- **Service Showcase** - Detailed presentation of company services
- **Project Gallery** - Portfolio of completed projects
- **Contact Integration** - Direct phone, email, and social media links
- **Scroll-to-Top** - Enhanced user experience

## 🏗️ Services Featured

- **Industrial Scaffolding** - Installation, dismantling, and rental
- **Suspended Scaffolding** - Specialized high-altitude access solutions
- **Thermal Insulation** - Energy efficiency optimization
- **Thermal Insulation Restoration** - Maintenance and replacement
- **Anti-corrosion Protection** - Durable metal structure treatments
- **Sandblasting & Anti-corrosion** - Industrial tank protection

## 🚀 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript** - Interactive functionality
- **Font Awesome** - Professional icons
- **Google Fonts** - Typography optimization
- **Responsive Design** - Mobile-first approach

## 📱 Responsive Breakpoints

- **Desktop**: 1200px+
- **Tablet**: 800px - 1199px
- **Mobile**: 600px and below
- **Small Mobile**: 480px and below

## 🎯 Key Sections

1. **Hero Section** - Company introduction and call-to-action
2. **Services** - Detailed service descriptions
3. **About Us** - Company history and expertise
4. **Recent Projects** - Success stories and case studies
5. **Why Choose Us** - Company advantages and experience
6. **Testimonials** - Client recommendations
7. **Contact Information** - Multiple contact methods

## 🔧 Setup Instructions

1. Clone the repository
2. Open `CIRASICO acasă.html` in your browser
3. Navigate to `proiectele-noastre.html` for the projects page
